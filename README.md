# MEMOME - frontend
> 방명록 

## 실행

```shell
npm install

npm run start:dev // dev 서버로 실행

npm run start:prod // prod 서버로 실행
```

`localhost:3000` 접속

## 사용

- React
- CSS3 & HTML
- javascript
